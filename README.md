# 3-D-Perspectives-of-Uttarakhand-from-Space
3-D Perspectives of Uttarakhand from Space” presents 3-D satellite views of Uttarakhand’s landscapes using Resourcesat-2/2A LISS-IV imagery and CartoDEM. Featuring 50 plates of mountains, valleys, and glaciers, it is valuable for researchers, mountaineers, GIS professionals, and tourism planners.
